# -IY4101-OOP-Java-Project-Abdullahi
This repository contains all code files for my Java Summative 3 Shapes project.

--- Project Overview

This project is a Java console-based Shape Management Application developed for the IY4101 Object Oriented Programming Practical Programming Assignment 2. The purpose of the application is to demonstrate core Object-Oriented Programming principles such as class design, encapsulation, inheritance, abstraction, polymorphism, method overriding, and object interaction.
The application allows users to create and manage different geometrical shapes, including rectangles, squares, circles, and triangles. These shapes can be added to a list, displayed, removed, translated, scaled, and used to calculate area and perimeter.

--- Features
The application provides the following menu options:

- Add a shape
- Remove a shape by position
- Get information about a shape by position
- Display the area and perimeter of a shape by position
- Display information about all shapes
- Translate all shapes
- Scale all shapes
- Quit the program

--- Classes Included:

-- Coordinates
-- Shape
-- Rectangle
-- Square 
-- Circle
-- Triangle
-- ShapeList
-- ShapeManagement

--- How to Run the Program
- Open the project in a Java IDE such as IntelliJ IDEA, Eclipse, VS Code, or NetBeans.
- Make sure all .java files are in the same project folder or package.
- Compile the Java files.
- Run the ShapeManagement class.
- Use the console menu to add, remove, display, translate, and scale shapes.
